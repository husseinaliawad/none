# Product Vision Blueprint

This document maps the requested product vision into concrete engineering phases.

## Implemented Foundation (This Iteration)

1. Performer profiles schema + performer-content links.
2. Strong tags schema (weighted tags + polymorphic links).
3. Storyboard metadata columns in both video tables.
4. Gamification foundation (rank levels + user progress).
5. Fan groups foundation.
6. Shorts foundation.
7. Personalized feed baseline service (tag-weight based ranking).

## Phase 1: Performer Experience

1. Admin CRUD for performers.
2. Link performers to videos/embedded videos.
3. Public performer pages with chronological filmography.

## Phase 2: AI Personalization

1. Real-time user behavior events (watch time, skips, repeats, likes).
2. Candidate retrieval by tags + embeddings.
3. Re-ranking model for personalized homepage rows.

## Phase 3: Storyboard and Clip Intelligence

1. FFmpeg jobs to generate storyboard sprite + VTT.
2. Timeline highlights extraction.
3. Auto-generated shorts from top segments.

## Phase 4: Community and Monetizable Retention

1. Rank perks gating by user level.
2. Performer fan groups (private/public tiers).
3. Erotic charts pages + daily/weekly trend boards.
